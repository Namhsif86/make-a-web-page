    • Basic Foundation of The Web and HTML

    The World Wide Web consist of HTML, Images, Videos, Pictures, Music, and Plain Text.

    The web mainly consist of HTML, which stands for HyperText Markup Language.

    The basic structure of the web is:
    -The Browser
    -The Internet- World's largest Cloud Network
    -HTTP - Main protocol of the web
    -Servers - They are computers designed to hold the searchable information.


    To create webpages with all the nice designs and embedded files, you must put HTML tags in < name > to specify
    what you want the computer to do with the webpage.
    Some Examples of HTML tags are:
    <b> - This tag makes everything BOLD
    <em> - This tag makes everything Italic
    <a> - This tag creates Hyperlinks
    <p> - This tag defines the area of a paragraph
    <br> - When using this tag, it creates a break
    so you go to the next line

    Their are plenty more complex tags to throw in at a later point.
    Tags can also be open <> or closed </>. When a tag is open <b>then the text after
    will remain in bold, only when</b>     you close the tag will the effects cease to continue.


    Adding CSS Style and Structure to Webpages.

    BOXES!!!
    -Webpages are made with multiple rectangular boxes.
    -The boxes are formed by:
    Margins
    Borders
    Padding
    Content
    -The size of the element is the sum of the border, padding, and content width.
    -CSS "box-sizing" is an easier way to define boxes.
    You can change the size of the box in either Pixels or Percentage

    CSS "Cascading Style Sheets"
    -CSS Allows programmers to add color and specific style's to webpages.
    -A programmer can either make a seperate CSS file or add the style to the HTML.
    When adding CSS sytle to the HTML code, the code must be in lt&stylegt& tags.

    DEV Tool's
    -The Developer Tool's in Chrome are a great tool for stutudent and professional's alike.
    -The tool allow's you to see the boxes, test out new dimensions, and styles.
